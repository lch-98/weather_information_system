## 🧵 Raspberry Pi Pico W + 실시간 기상청 Weather API

이 저장소는 **Raspberry Pi Pico W**에서 **실시간 기상청 Weather API**를 받아와 **실시간 기상 정보**를 LCD에 표시하는 1인 프로젝트입니다.

(`FreeRTOS, SPI, Http, Wifi 등`)의 기술을 **실습**해 볼 수 있습니다.
<br><br>

## 🛠️ 개발 환경

- Ubuntu >= 20.04
- Raspberry Pi Pico W
- FreeRTOS Kernel
- Raspberry Pi Pico SDK
- CMake >= 3.13
- arm-none-eabi-gcc
- VScode
